优选IP订阅生成器

导入自建节点 → 批量替换为优选 IP（可选） → 输出可导入主流客户端的订阅链接。
支持 VMess / VLESS / Trojan 的 WS + TLS 常见配置。

📝 支持格式
  • IP 或 IP:端口
  • 域名 或 域名:端口
  • # 后面写备注
  • 留空则纯格式转换

支持多行粘贴，也支持 Base64 编码的订阅内容

格式：104.16.1.2#备注 或 cdn.example.com:2053#美国，留空时不替换地址

💡 说明
  • 不会帮你查找优选 IP，只负责替换进节点
  • 推荐勾选"保留原 Host / SNI"，适合 CDN 场景
  • 不填优选地址时，仅做格式转换（不替换服务器地址）
  • 订阅链接 7 天后自动过期
